# AGENT.md (Folder-Level Guide)

## Purpose of this folder
Build scripts, automation tools, and utility scripts for repository operations.

## What agents may do here
- Create utility scripts for automation
- Add build and deployment scripts
- Create tools for code generation or validation
- Follow Node.js scripting best practices
- Document script usage clearly

## What agents may NOT do
- Create scripts that bypass security checks
- Hardcode secrets or credentials
- Create scripts with destructive operations without safeguards
- Bypass governance verification

- Refer to higher-level policy: /.repo/policy/SECURITY_BASELINE.md
- Automation stubs: /.repo/automation/
