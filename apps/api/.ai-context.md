# API Server

This is the Express/Fastify API server.

## Framework
- Express or Fastify
- Middleware in `middleware/`
- Routes in `routes.ts`
- Storage utilities in `utils/`

## Structure
- `index.ts` - Entry point
- `middleware/` - Auth, error handling, rate limiting, validation
- `routes.ts` - API routes
- `storage.ts` - Storage utilities
- `utils/` - API utilities
